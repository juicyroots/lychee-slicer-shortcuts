# File Operations

Command | Shortcut
------------ | -------------
New Scene | '<CTRL>' + '<N>'
Import a File | <CTRL> + <I>
Open a File | <CTRL> + <O>
Export All | <CTRL> + <E>
Export Selected | <CTRL> + <>
Save Scene As | <CTRL> + <>
Show Preferences | <CTRL> + <>
Reload Lychee | <CTRL> + <>
Save Scene | <CTRL> + <>
